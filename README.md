# Market Kit Skills

面向真实营销交付的 AI skill 仓库。

这个仓库不是放一堆零散 prompt 的地方，而是用来分发一套真正能把营销需求推进成结果的 skill。当前核心内容是：

- `skills/market-kit-skills`

## 这个仓库里的 skill 能做什么

- 输出营销方案、内容规划和 campaign plan
- 生成小红书图文、小红书笔记、标题、正文和配图
- 提炼人群、卖点、定位和差异化表达
- 基于资料库生成参考驱动内容
- 返回文案、图片链接和网页版结果链接，便于继续追踪和复用

## 为什么值得装

因为它不是只会说几句漂亮话，而是更接近一条完整的营销生成链路。

你可以拿它做：

- 新品营销方案
- 品牌种草内容
- 小红书图文内容
- 卖点和人群梳理
- 图文一体的营销交付物

## 安装方式

安装方式很简单：把这个 GitHub 仓库地址发给你的 Agent，然后告诉它帮你安装 `market-kit-skills` 就可以。

```text
请帮我安装这个 skill：
https://github.com/qinshimeng18/xiaojia-skills

skill 名称：
market-kit-skills
```
